A starter theme called Palmixio.

== Description ==

An old theme for WordPress, very simple with new restyling and responsive, 2 column layout, custom header, custom logo and compatible for use with Font Awesome Icons.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
