**Easily manage your video courses, webinars, and content for your WordPress users to watch, track, comment and follow.**

The Video Courses Manager plugin lets users track and organize online video courses and webinars on your WordPress site.

[vimeo https://vimeo.com/203769044]

The Video Lessons Manager plugin allows you to manage video courses, lectures and lessons while users and administrators can track progress, leave comments and mark favorites. You can set up online video-driven courses by creating learning units supported by course content.			

The Video Courses Manager plugin allows you to manage video courses, video tutorials and lessons while users and administrators can track progress, leave comments and mark favorites. This plugin creates a WordPress learning management system where uses can check course progress, and video content.

The pro version of the plugin also supports adding pay-per-view lessons for limited time periods while using direct payment option and checkout with PayPal, Stripe and more or Micropayment solution.	

The pro plugin supports watching videos generated on Vimeo Private Video Channels or Wistia. Build several courses or lessons based on your Vimeo or Wistia account. Display your videos in any page or post on your WordPress site.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Manage your CM Video Lessons from Left Side Admin Menu
